## 1. 游戏概述

### 1.1 游戏定位
单机弱联网挂机养成游戏，以“轮回”为核心机制，玩家通过不断轮回积累永久强化，逐步解锁更深层玩法。

### 1.2 核心体验
- 每次轮回都比上一次更强
- 挂机与主动操作结合
- 多维度养成路径
- 长期目标驱动

### 1.3 设计原则
> **“只要比上次有进步，哪怕只有50或100级，也不要花几倍的时间去升200级。”**

## 2. 核心玩法循环

```
┌─────────────────────────────────────────────────────────┐
│                    核心玩法循环                          │
├─────────────────────────────────────────────────────────┤
│                                                         │
│   ┌─────────┐    ┌─────────┐    ┌─────────┐           │
│   │  开局   │───→│ 工作赚钱 │───→│ 研究升级 │           │
│   └─────────┘    └─────────┘    └─────────┘           │
│        ↑              │              │                  │
│        │              ↓              ↓                  │
│   ┌─────────┐    ┌─────────┐    ┌─────────┐           │
│   │  轮回   │←───│ 战斗防御 │←───│ 生活消费 │           │
│   └─────────┘    └─────────┘    └─────────┘           │
│        │              │              │                  │
│        ↓              ↓              ↓                  │
│   ┌─────────────────────────────────────────┐          │
│   │         永久成长 (穿越者/科技树/粒子)      │          │
│   └─────────────────────────────────────────┘          │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

### 2.2 阶段玩法说明

| 阶段   | 时间范围 | 核心目标           | 策略重点                   |
| ------ | -------- | ------------------ | -------------------------- |
| 开荒期 | 1-10轮回 | 理解机制，积累基础 | 小步快跑，反复轮回         |
| 黑奴期 | 前50岁   | 快速赚钱           | 全部时间工作，最低生活标准 |
| 冲刺期 | 50-70岁  | 拉高研究等级       | 全部时间研究，最高生活标准 |
| 防御期 | 70-90岁  | 战斗防御           | 建造防御炮台，抵御怪物     |
| 轮回期 | 90岁+    | 主动/被动轮回      | 结算资源，开启下一轮       |

## 3. 属性系统

### 3.1 六维属性

| 属性 | 英文       | 基础值 | 每点效果              | 计算公式                               |
| ---- | ---------- | ------ | --------------------- | -------------------------------------- |
| 心态 | Mindset    | 10     | +0.5%精力             | 精力加成 = 1 + (心态-100)×0.005        |
| 健康 | Health     | 10     | +0.01基础精力         | 基础精力 = 1 + (健康-100)×0.0001       |
| 智商 | IQ         | 10     | +1%研究经验           | 研究加成 = 1 + (智商-100)×0.01         |
| 情商 | EQ         | 10     | +1%工作经验/收入      | 工作加成 = 1 + (情商-100)×0.01         |
| 创意 | Creativity | 10     | -0.1%研究所需经验     | 经验减免 = min(99%, (创意-100)×0.001)  |
| 运气 | Luck       | 10     | +1w初始资金/+0.1%掉率 | 掉率加成 = min(100%, (运气-100)×0.001) |

> **注：属性只计算超过100的部分，基础值10无实际加成效果**

### 3.2 精力系统

```
精力 = 基础精力 × (1 + 心态加成) × 其他加成

基础精力 = 1 + 健康加成 + 穿越者加成 + 科技树加成
```

精力影响：
- 工作升级效率
- 研究升级效率

## 4. 时间分配系统

### 4.1 每日时间分配

```typescript
interface TimeAllocation {
  sleep: number;    // 睡眠时间 (0-24)
  work: number;     // 工作时间 (0-24)
  research: number; // 研究时间 (0-24)
  // 约束: sleep + work + research = 24
}
```

### 4.2 推荐分配策略

| 策略     | 睡眠 | 工作  | 研究  | 适用场景         |
| -------- | ---- | ----- | ----- | ---------------- |
| 黑奴策略 | 0    | 23:15 | 0     | 前50岁赚钱期     |
| 研究策略 | 8    | 0     | 16    | 50岁后研究期     |
| 平衡策略 | 8    | 8     | 8     | 新手期           |
| 防御策略 | 8    | 5-6   | 10-11 | 70岁后战斗准备期 |

### 4.3 时间消耗规则

- 吃饭时间：15分钟（可被穿越者技能减免）
- 交通时间：根据交通工具不同（单车90分钟/防弹车30分钟）
- 住宿时间：0分钟（雇佣清洁工后）

## 8. 轮回系统

### 8.1 轮回类型

| 类型     | 触发条件                         | 说明                   |
| -------- | -------------------------------- | ---------------------- |
| 被动轮回 | 母星被怪物攻克                   | 自动触发，结算当前进度 |
| 主动轮回 | 解锁研究【超时空轮回】后手动点击 | 可主动结束当前轮回     |

### 8.2 轮回结算资源

| 资源     | 获取公式                             | 用途                  |
| -------- | ------------------------------------ | --------------------- |
| 记忆因子 | 工作最高等级×10 + 研究最高等级×5     | 升级【时间弹性】      |
| 时间结晶 | 消耗加速天数 ÷ 2190 (满级时空旅人后) | 升级科技树            |
| 高维核心 | 消耗加速天数 ÷ 2920 (满级高维搜索后) | 升级科技树(战斗/星空) |
| 轮回勋章 | 完成任务数量                         | 商店兑换              |

### 8.3 轮回保留内容

```
✅ 保留：
- 科技树等级
- 穿越者等级
- 粒子升级
- 量子币
- 部分道具

❌ 重置：
- 等级
- 金钱
- 年龄
- 时间分配
- 建筑（部分保留）
```

### 8.4 速刷轮回策略

```yaml
速刷条件:
  - 有充足加速时间
  - 时空旅人1-4已满级
  - 高维置换已解锁
  - 高维搜索1-2已解锁

速刷操作:
  1. 选择出生事件
  2. 点击【时间跃迁】
  3. 生存到43岁左右
  4. 主动轮回

原理: 加速天数消耗达到2190倍数时获取时间结晶
```

### 15.2 核心建筑

| 建筑     | 功能     | 升级效果                      | 优先级 |

| 星舰平台 | 星舰建造 | 解锁更高级星舰                | 中     |
| 星舰港口 | 星舰停泊 | +停泊数量                     | 中     |

### 15.3 文明系统

```
文明发展:
├── 文明建筑 (产出哈希值/文化值)

│   ├── 主权宣布 (解锁殖民)
│   ├── 进阶强化 (解锁高级升级)
│   └── ...
└── 联盟兑换 (消耗联盟币)
```

```yaml
  - 轮回结束时选择带入

  - 星舰被摧毁需重新建造或从分支带出
```

## 21. 技术架构建议

### 21.1 前端技术栈

```yaml
框架: React 18+ / Vue 3+
语言: TypeScript
状态管理: Zustand / Redux Toolkit
UI库: Ant Design / Tailwind CSS
游戏循环: requestAnimationFrame + setInterval fallback
存储: IndexedDB (localForage) + localStorage
```

### 21.2 后端 (如需联网功能)

```yaml
框架: Node.js + Express / NestJS
数据库: PostgreSQL + Redis
认证: JWT
实时同步: WebSocket
```

### 21.3 核心数据结构


### 21.4 性能优化建议

1. **游戏循环节流**：使用 `useReducer` + `requestAnimationFrame`
2. **状态持久化**：自动保存（每30秒）+ 手动保存
3. **计算缓存**：使用 `useMemo` 缓存复杂计算
4. **懒加载**：按需加载非核心模块
5. **离线支持**：Service Worker + IndexedDB
